# Scientific Calculator

A web-based scientific calculator application built with HTML, CSS, and JavaScript.

## Features

- **Standard Operations**: Addition, subtraction, multiplication, division, and exponentiation.
- **Scientific Functions**: 
  - Trigonometric: `sin`, `cos`, `tan`, `sin⁻¹`, `cos⁻¹`, `tan⁻¹`
  - Logarithmic: `log` (base 10), `ln` (natural log)
  - Others: `sqrt` (square root), `abs` (absolute value), `!` (factorial)
- **Constants**: Supports `π` (pi), `e` (Euler's number), and `Ans` (previous result).
- **Calculator Modes**: Toggle between Degrees (DEG) and Radians (RAD) for trigonometric functions.
- **Keyboard Support**: Perform calculations directly using your keyboard.
- **Responsive Display**: Cleans up and formats input expressions.

## Getting Started

1. Clone this repository:
   ```bash
   git clone <repository-url>
   ```
2. Open `index.html` in your web browser.

## How to use

- **Click buttons** on the interface or **type directly** into the calculator using your keyboard.
- Use **`C`** to clear the display.
- Use **`⌫`** to delete the last character.
- Press **`=`** or **`Enter`** to calculate the result.
